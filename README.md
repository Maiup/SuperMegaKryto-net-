## 🎲 Random Task Generator

Консольное приложение на Python для генерации случайных задач с фильтрацией, историей и сохранением данных в JSON.

### 📖 Краткое описание

**Random Task Generator** помогает генерировать случайные задачи для планирования дня, тренировок, учёбы или работы. Программа использует объектно-ориентированный подход и паттерн **Factory** для создания задач разных типов.

### Основные возможности:

- ✅ **Генерация случайной задачи** из предустановленного списка
- 🔍 **Фильтрация задач** по типу (Работа/Спорт/Учёба) и сложности (1–5)
- ➕ **Добавление собственных задач** с валидацией ввода
- 📜 **История последних 10 задач** (реализована через очередь `deque`)
- 💾 **Автосохранение** всех задач и истории в JSON-файл
- 🎨 **Цветовое выделение** сложности звёздочками (★☆☆☆☆)

### Технологии:

- Чистый Python 3.7+
- Только стандартные библиотеки: `json`, `random`, `collections`, `abc`
- Объектно-ориентированный дизайн (инкапсуляция, наследование, полиморфизм)
- Паттерн **Factory Method** для создания задач
- Хранение истории в очереди с ограничением в 10 элементов
